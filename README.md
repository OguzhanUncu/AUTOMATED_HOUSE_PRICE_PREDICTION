# AUTOMATED HOUSE PRICE PREDICTION

![iStock-1218603907-770x513](https://user-images.githubusercontent.com/76595310/127649209-af84df27-19ca-4c6c-aef8-773fe7ed9be2.jpg)

## Business Problem

* It is required to create a price estimation model on houses with different characteristics.

## Story of Dataset

* This dataset of residential homes in Lowa, Ames includes 79 explanatory variables.

* There are two different csv files, train and test.

* House prices are NaN in the test dataset and we are expected to estimate these values.
